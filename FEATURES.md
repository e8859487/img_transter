# 功能列表

## 📁 自動監控與轉移
自動監控來源資料夾，每 5 秒掃描一次新增的圖片與影片檔案。
支援跨磁碟區轉移（例如外接 USB 硬碟），並自動驗證檔案完整性。

## 📅 智慧分類整理
根據檔案的最後修改時間，自動建立「年/月」資料夾結構。
轉移後的檔案會整齊地分類到對應的時間資料夾中。

## 🗑️ 彈性刪除選項
提供「轉移後自動刪除來源檔案」的開關選項。
若未開啟自動刪除，可稍後使用「一鍵刪除來源」按鈕批次清理。

## 📊 即時進度顯示
轉移檔案時顯示當前檔案名稱與進度百分比。
使用視覺化進度條即時呈現轉移狀態，讓使用者清楚掌握進度。

## 📝 詳細轉移日誌
記錄所有轉移操作的時間戳記與檔案名稱。
支援清除日誌功能，最多保留 200 筆記錄。

## 💾 路徑記憶功能
自動儲存上次選擇的來源與目標資料夾路徑。
下次開啟程式時自動載入，並驗證路徑是否仍然有效。

## 🎯 智慧檔案過濾
僅處理圖片（JPG、PNG、HEIC 等）與影片（MP4、MOV 等）格式。
自動跳過隱藏檔案、暫存檔案，以及正在寫入中的檔案。

## 🔒 安全性保障
採用檔案大小比對機制，確保檔案完全寫入後才開始轉移。
轉移完成後驗證目標檔案大小，確保資料完整無誤。

## 🤖 Agent 自動化管理
具備 AI 助手自動化規則，確保開發過程中功能說明文件永遠即時更新。
包含標準化的功能開發流程與工作流，提升專案維護的效率與一致性。
